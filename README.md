# free-sup

wusfe 的 Claude Code 插件市场。

## 安装

```bash
# 添加市场
/plugin marketplace add wusfe/free-sup

# 安装插件
/plugin install <plugin-name>@free-sup
```

## 插件

| 插件 | 说明 |
|------|------|
| [plugin-forge](./plugins/plugin-forge/) | 从外部市场搜索、评估并导入 Claude Code 插件到 free-sup 市场。支持搜索候选、改造格式、安全检查、提交 PR 的全流程。 |


## 贡献

欢迎提交 PR 贡献新插件或改进现有插件。插件需包含 `.claude-plugin/plugin.json` 配置文件。
