# /claude-hud:configure

Claude HUD 状态栏的交互式配置命令。

## 触发方式

- 用户在 Claude Code 会话中输入 `/claude-hud:configure`
- 用户想要自定义 HUD 显示效果

## 流程 A — 新用户（6个问题）

1. **布局** — 选择显示布局：`expanded`（默认，多行）或 `compact`（单行）
2. **预设** — 三种预设：
   - `Full`：全部开启（工具、代理、待办、环境）
   - `Essential`：活动行 + git 状态
   - `Minimal`：仅模型名 + 上下文条
3. **语言** — 选择界面语言：`en`（英文）或 `zh`（中文）
4. **元素** — 开关单个显示元素
5. **自定义行** — 底部可选的自定义文本行
6. **预览** — 显示最终布局的文本预览

## 流程 B — 老用户

1. **开关元素** — 开启/关闭特定元素
2. **Git 样式** — 选择 git 状态显示样式
3. **布局/重置** — 更改布局或恢复默认
4. **语言** — 更改界面语言
5. **自定义行** — 设置或修改自定义行
6. **预览** — 应用前预览

## 输出

写入 `~/.claude/plugins/claude-hud/config.json`，重启 Claude Code 后生效。
